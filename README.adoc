= docker-node-red

Packaged Node-Red for use in home automation.

== Build

[source,bash]
----
$ docker build -t kalemena/node-red .
----

== Run with compose

Run with opening 1880 port, and map your flows into /data directory.

Optionaly, map USB devices used in flows.

[source,bash]
.docker-compose.yml
====
include::docker-compose.yml[]
====

=== Start node-red

[source,bash]
----
$ docker-compose up -d
----

Now you can connect on *http://&lt;yourhost&gt;:1880*

=== Stop node-red

[source,bash]
----
$ docker-compose stop
----

=== Remove container node-red

[source,bash]
----
$ docker-compose down
----
